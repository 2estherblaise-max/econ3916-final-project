# ECON 3916 Final Project — Credit Default Prediction

Predicting credit card default using the UCI Default of Credit Card Clients dataset (N=30,000).

## Dataset
UCI ML Repository — Default of Credit Card Clients (ID=350)
https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients

## Models
- Logistic Regression (baseline)
- Random Forest (final submission)

## To reproduce
1. Open `econ3916_final_checkpoint.ipynb` in Google Colab
2. Run all cells
